# bookmarker

## 다운로드
1. bookmarker.zip 파일 다운로드
2. 압축해제

## 설치
1. 크롬 브라우저에서 "확장 프로그램 관리" 열기
2. 상단에 "개발자 모드" 체크 (활성화)
3. "압축해제된 확장 프로그램 로드" 버튼 클릭
4. bookmarker.zip 파일을 압축해제한 폴더 선택

## 실행
1. Bookmarker AI Manager 1.0.0 확장 선택 후 "세부정보" 버튼 클릭
2. "툴바에 고정" 체크 (활성화)
3. 툴바에서 Bookmarker 클릭(파란색 B 아이콘)
4. 하단에 "설정" 클릭
5. 기본 설정: 언어, AI 기능, AI 설정, 카테고리 등 설정
6. 기본 설정을 하지 않고 바로 사용하면 키워드 검색만 가능

## 북마크 등록 및 분석
1. 툴바에서 Bookmarker 클릭
2. 하단에 "대시보드" 클릭
3. 대시보드 화면에서 "일괄 등록 시작" 버튼 클릭 : 북마크를 등록하고 카테고리 분류 및 태그를 생성함 
4. 대시보드 화면에서 "일괄 분석 시작" 버튼 클릭 : 중복 등록된 북마크 삭제함

## 북마크 검색
1. 툴발에서 Bookmarker 클릭
2. 검색창에 찾고자하는 북마크의 단어 입력
3. 검색 결과 확인 후 항목 클릭 시 페이지 열림
